# Contributing to NewsSense

Thank you for your interest in contributing to NewsSense! However, this is a university project and currently not accepting external contributions.

## For Collaborators
If you're part of the development team, please:
1. Create a feature branch for your changes
2. Test your code thoroughly before submitting a pull request
3. Follow the existing code style
